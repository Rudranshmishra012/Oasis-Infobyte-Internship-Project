# Android App Market Analysis

## About the Project

In this project, I analyzed data from the Google Play Store to understand trends in app categories, ratings, installs, pricing, and content ratings. The goal was to explore the Android app market and gain insights using Python and data analysis techniques.

## Dataset

The dataset contains information about thousands of Android applications available on the Google Play Store, including:

* App Name
* Category
* Rating
* Reviews
* Size
* Number of Installs
* Type (Free/Paid)
* Price
* Content Rating
* Genre
* Last Updated Date

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook / VS Code

## Data Cleaning

Before starting the analysis, I cleaned the dataset by:

* Handling missing values in important columns such as Rating, Size, Current Version, and Android Version.
* Converting columns like Installs and Price into numeric formats for analysis.
* Converting the Last Updated column into datetime format.
* Removing unnecessary columns.
* Checking for duplicate records.

## Analysis Performed

### Top App Categories

Analyzed which categories have the highest number of applications on the Play Store.

### Free vs Paid Apps

Compared the distribution of free and paid applications.

### Rating Distribution

Studied how app ratings are distributed across the dataset.

### Average Rating by Category

Identified app categories with the highest average user ratings.

### Most Installed Categories

Analyzed which categories have the highest total number of installs.

### Paid App Price Analysis

Explored the pricing pattern of paid applications.

### Content Rating Analysis

Examined the distribution of apps based on content ratings such as Everyone, Teen, and Mature.

## Key Insights

* Most applications available on the Play Store are free.
* Family, Game, and Tools are among the most popular categories.
* Most apps have ratings between 4.0 and 4.5.
* Some categories attract significantly more installs than others.
* Paid apps represent a small portion of the market.
* The majority of apps are suitable for all age groups.
