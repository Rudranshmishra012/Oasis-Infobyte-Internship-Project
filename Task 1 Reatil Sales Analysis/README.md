# Retail Sales Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a Retail Sales Dataset using Python and Pandas. The objective is to analyze customer behavior, product performance, and sales trends to generate meaningful business insights.

## Objectives

* Understand customer purchasing patterns.
* Analyze product category performance.
* Identify revenue-generating categories.
* Explore demographic trends.
* Visualize sales data using charts and graphs.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* VS Code

## Dataset Information

The dataset contains 1000 retail transactions with the following attributes:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

## Data Cleaning

* Checked for missing values.
* Checked for duplicate records.
* Converted date column into datetime format for analysis.

## Exploratory Data Analysis

### 1. Product Category Distribution

Analyzed the number of transactions across different product categories.

### 2. Gender Distribution

Examined the proportion of male and female customers.

### 3. Age Distribution

Studied customer age patterns using a histogram.

### 4. Revenue by Product Category

Calculated total revenue generated by each product category.

### 5. Monthly Sales Trend

Analyzed sales performance over time.

## Key Findings

* No missing values were found in the dataset.
* No duplicate records were found.
* Clothing had the highest number of transactions.
* Electronics generated the highest revenue (156,905).
* Clothing generated revenue of 155,580.
* Beauty generated revenue of 143,515.
* Male and female customers were almost equally distributed.
* The average customer age was approximately 41 years.

## Business Insights

* Electronics contributes the highest revenue despite not having the highest number of transactions.
* Clothing is the most frequently purchased category.
* Customer purchases are well distributed across genders.
* The business can focus on Electronics and Clothing categories to maximize revenue growth.

## Conclusion

The analysis provides valuable insights into customer demographics, product performance, and revenue generation. These findings can help businesses make data-driven decisions regarding inventory management, marketing strategies, and sales optimization.
